Views and it's properties


                In NX (Unigraphics NX), view properties refer to the settings and characteristics associated with the different views
                you create on a drawing sheet. These properties allow you to control the appearance and behavior of the views, ensuring
                that they convey the necessary information accurately. Here are some key aspects of view properties in UG NX:
                
                View Scale: 
                The scale of a view determines the size at which the 3D model is represented on the drawing sheet. You can set the scale
                based on factors such as the level of detail required and the available space on the sheet.
                
                View Display:
                UG NX provides options for displaying or hiding certain elements within a view. This includes features like hidden
                lines, tangent edges, and other visibility settings that help communicate the design intent effectively.
                
                View Type:
                Different types of views, such as orthographic, isometric, and detailed views, can be created in UG NX. Each view type
                has specific properties and settings that you can customize based on your drawing requirements.
                
                Section Views:
                If you create section views to show internal details of a part, you can adjust properties such as the cutting plane,
                section depth, and hatching style. This helps in clearly presenting the sectional information.
                
                Detail Views:
                Detail views focus on specific areas of the drawing at an enlarged scale. You can customize properties such as the
                detail view scale factor and the appearance of the detail boundary.
                
                Projected Views:
                Projected views are additional views created from existing views. You can control properties like the projection
                direction and the alignment of projected views.
                
                View Labels and Annotations:
                UG NX allows you to add labels and annotations to views, providing information such as view names, scale, and other
                relevant details. These annotations can be customized based on your drafting standards.
                
                View Alignment and Arrangement:
                You can control the alignment and arrangement of views on the drawing sheet. UG NX provides tools to organize views in a
                logical and readable manner.
                
                Associativity:
                Views in UG NX are typically associative to the 3D model. This means that if the 3D model is updated, the associated
                views on the drawing sheet can be automatically updated to reflect the changes.
                Accessing and modifying view properties in UG NX is typically done through the drafting environment. You can select a
                view, right-click, and access the properties dialog or use specific drafting commands to customize view settings.

Nx Code

            using NXOpen;

            namespace Views

            {
                public class view
                {
                   public static void Main(string[] args)
                    {
                        Session theSession = Session.GetSession();
                        Part workPart = theSession.Parts.Work;
            
                        NXOpen.Drawings.DrawingSheet DwgSheet = workPart.DrawingSheets.CurrentDrawingSheet;
                        NXOpen.View[] Views = DwgSheet.GetDraftingViews();
            
                        foreach(NXOpen.View sView in Views)
                        {
                            sView.SetScale(10.0);  // Change scale of all views to 2:1
                              
                        }
                    }
            
                    public static int GetUnloadOption(string args)
                    {
                        return (int)NXOpen.Session.LibraryUnloadOption.Immediately;
                    }
                }
            }
