Drawings

        Key aspects of drawing in NX OPEN include:
        
        Views: You can generate orthographic or isometric views of the 3D model to show different perspectives of the object.
        These views help convey the shape, size, and features of the design.
        
        Dimensions: UG NX allows you to add accurate dimensions to the drawing, specifying distances, angles, and other
        measurements. This information is crucial for manufacturing and quality control.
        
        Annotations: You can add annotations, notes, and labels to provide additional information or instructions. This helps in
        clarifying specific details or requirements.
        
        GD&T (Geometric Dimensioning and Tolerancing): UG NX supports GD&T, allowing you to define and communicate tolerances
        for various geometric features. This is essential for ensuring that the manufactured part meets the intended design
        specifications.
        
        Bill of Materials (BOM): UG NX enables the creation of a Bill of Materials, listing the components and quantities
        required for assembly. This information is vital for manufacturing and procurement processes.
        
    
        A drawing is represented by a collection of NXOpen.Drawings.DrawSheet objects in NX Open. The set of all DrawingSheet
        objects in the work part (or any part file) is a DrawingSheetCollection object, which you can get by using the
        workPart.DrawingSheets property.

        Each sheet has a SheetDraftingViewCollection object, which is important because you use it to work with the views on the
        sheet (to create and delete views, for example). You can get this object by using the SheetDraftingViews property of the
        sheet.

Sample Code

            using NXOpen;

            namespace Drawing

            {
                public class Drawing
                {
                    public static void Main(string[] args)
                    {
                        Session theSession = Session.GetSession();
                        Part workPart = theSession.Parts.Work;
            
                        NXOpen.Drawings.DrawingSheet DwgSheet = workPart.DrawingSheets.InsertSheet("S01", NXOpen.Drawings.DrawingSheet.StandardSheetSize.A3, 1, 1, NXOpen.Drawings.DrawingSheet.ProjectionAngleType.ThirdAngle);
                        DwgSheet.Open();
            
                    }
            
                    public static int GetUnloadOption(string args)
                    {
                        return (int)NXOpen.Session.LibraryUnloadOption.Immediately;
                    }
                }
            }
