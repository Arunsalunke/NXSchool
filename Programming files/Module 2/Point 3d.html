Point 3d Object


        In the context of Siemens NX (formerly Unigraphics or UG), Point3D objects are used to represent points in a
        three-dimensional space within the NX Open API.

        The Point3D class represents a point in 3D space and is part of the NX Open namespace. It contains X, Y, and Z
        coordinates to define a point's position.

Example

            using NXOpen;

            {

                 {

                        Session theSession = Session.GetSession();

                        ListingWindow lw = theSession.ListingWindow;
                        
                        Point3d p1 = new Point3d(10, 20, 30);

        The above code shows the creation of a Point3D object with coordinates (10.0, 20.0, 30.0). It demonstrates how to access
        and manipulate the coordinates of the point.

        In NX Open, these Point3D objects are commonly used when dealing with geometric entities such as curves, surfaces, or
        bodies to specify positions, directions, or locations for various operations or transformations within the software.

        objects are gone. In this sense, they are just like the numerical variables that you use in your programs. If you want
