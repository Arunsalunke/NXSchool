String

        In C#, strings are sequences of characters represented by the string data type. Strings are widely used for working with
        text and are part of the System.String class in the .NET framework. Here are some key aspects of working with strings in
        C#:

1. String Declaration:


        string myString = "Hello, World!";

2. String Concatenation:


        You can concatenate strings using the + operator:

        string firstName = "John";
        string lastName = "Doe";
        string fullName = firstName + " " + lastName;

        Alternatively, you can use the String.Concat() method or string interpolation:
        
        string fullName = string.Concat(firstName, " ", lastName);
        // or
        string fullName = $"{firstName} {lastName}";

3. String Length:


        The Length property is used to get the number of characters in a string:

        string myString = "Hello, World!";
        int length = myString.Length; // length is 13

4. Accessing Characters:


        You can access individual characters in a string using indexing:
        char firstChar = myString[0]; // firstChar is 'H'

5. String Comparison:


        Use methods like Equals() or Compare() for string comparison:

        string str1 = "hello";
        string str2 = "HELLO";
        bool areEqual = str1.Equals(str2, StringComparison.OrdinalIgnoreCase);

6. Substring:


        Extract a portion of a string using Substring():
        
        string original = "Hello, World!";
        string substring = original.Substring(0, 5); // substring is "Hello"

7. String Manipulation:


        Various methods are available for manipulating strings, such as ToLower(), ToUpper(), Trim(), Replace(), Split(), etc.

        string original = " Trim Me ";
        string trimmed = original.Trim(); // trimmed is "Trim Me"

8. String Formatting:


        You can format strings using the String.Format() method or string interpolation:

        int age = 25;
        string message = String.Format("My age is {0}.", age);
        // or
        string message = $"My age is {age}.";

9. String Comparison Operators:


        You can use == and != to compare strings:
        string str1 = "hello";
        string str2 = "world";
        bool areEqual = (str1 == str2); // false

10. Immutable Nature:

  
        Strings in C# are immutable, meaning that once a string object is created, its value cannot be changed. Any operation
        that seems to modify a string actually creates a new string.
        
        These are just some of the basics of working with strings in C#. The System.String class provides a rich set of methods
        for manipulating and working with text data. Additionally, C# provides the StringBuilder class for more efficient string
        manipulation when dealing with large amounts of text.
