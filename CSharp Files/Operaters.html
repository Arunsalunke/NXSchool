Operators

        In C#, operators are symbols or keywords that perform operations on variables and values. C# supports a variety of
        operators, which can be categorized into several groups based on their functionality. Here's an overview of the main
        types of operators in C#:

There are six types operators


        1. Arithmetic Operators
        2. Relational Operators
        3. Logical Operators
        4. Assignment Operators
        5. Increment and Decrement Operators
        6. Conditional Operator

1. Arithmetic Operators


        1. + (Addition): Adds two operands.  
        int result = 5 + 3; // Result is 8

        2. - (Subtraction): Subtracts the right operand from the left operand.
        int result = 7 - 4; // Result is 3

        3. * (Multiplication): Multiplies two operands.
        int result = 6 * 2; // Result is 12


        4. / (Division): Divides the left operand by the right operand.
        int result = 10 / 2; // Result is 5

        4. % (Modulus): Returns the remainder of the division of the left operand by the right operand.
        int result = 13 % 5; // Result is 3

        bool isEqual = (5 == 5); // isEqual is true

        2. != (Inequality): Checks if the values of two operands are not equal.
        bool isNotEqual = (7 != 5); // isNotEqual is true

3.Logical Operators


        1. && (Logical AND): Returns true if both operands are true.
        bool result = (true && false); // Result is false

        2. || (Logical OR): Returns true if at least one operand is true.
        bool result = (true || false); // Result is true

        3. ! (Logical NOT): Returns true if the operand is false, and vice versa.
        bool result = !true; // Result is false

4.Assignment Operators:


        1. = (Assignment): Assigns the value on the right to the variable on the left.
        int x = 10; // Variable x is assigned the value 10

        2. +=, -=, *=, /=, %= (Compound Assignment): Performs the operation and assigns the result to the left operand.
        int x = 5;
        x += 3; // Equivalent to x = x + 3; (x is now 8)

5.Increment and Decrement Operators


        1. ++ (Increment): Increases the value of the operand by 1.
        int x = 5;
        x++; // Equivalent to x = x + 1; (x is now 6)

        2. -- (Decrement): Decreases the value of the operand by 1.
        int x = 5;
        x--; // Equivalent to x = x - 1; (x is now 4)
