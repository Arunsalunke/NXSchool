User Input

        In C#, you can read user input from the console using the Console.ReadLine() method. This method reads a line of
        characters from the standard input (console) and returns it as a string.
        
        Here's a simple example of reading user input in C#:
    
            using System;
            
            class Program
            {
                static void Main()

                {
                    Console.Write("Enter your name: ");
                    string userName = Console.ReadLine();
                    
                    Console.WriteLine($"Hello, {userName}!");
                    
                    // Wait for user to press Enter before closing the console window
                    Console.ReadLine();
                   
                }      
            }

        
        In this example:
        
        The Console.Write("Enter your name: "); statement prompts the user to enter their name.

        Console.ReadLine() reads the user's input as a string and assigns it to the userName variable.

        The program then prints a greeting using the entered name with Console.WriteLine($"Hello, {userName}!");.

        Console.ReadLine(); is used to wait for the user to press Enter before closing the console window.

        It's important to note that Console.ReadLine() returns the user input as a string. If you need to process the input as a
        different data type (e.g., convert it to an integer or a double), you'll need to perform type conversion using methods
        like int.Parse() or double.Parse().
        
    
Here's an example that reads an integer from the user:
            using System;

            class Program

            {
                static void Main()
                {
                    Console.Write("Enter your age: ");
                    string ageInput = Console.ReadLine();
            
                    // Convert the input to an integer
                    if (int.TryParse(ageInput, out int age))
                    {
                        Console.WriteLine($"You entered: {age}");
                    }
                    else
                    {
                        Console.WriteLine("Invalid input. Please enter a valid integer.");
                    }
            
                    Console.ReadLine(); // Wait for user to press Enter
                }
            }
            


        
        In this example, int.TryParse() is used to attempt parsing the user's input as an integer. If the conversion is
        successful, it prints the entered age; otherwise, it displays an error message.
