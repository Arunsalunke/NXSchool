Type Casting

        Type casting in C# is the process of converting a variable from one data type to another. C# supports two main types of
        casting: implicit casting (also known as widening conversion) and explicit casting (also known as narrowing conversion).
        The choice between implicit and explicit casting depends on whether the conversion is considered safe or might result in
        data loss or unexpected behavior.

1. Implicit Casting (Widening Conversion):

        Type casting in C# is the process of converting a variable from one data type to another. C# supports two main types of
        casting: implicit casting (also known as widening conversion) and explicit casting (also known as narrowing conversion).
        The choice between implicit and explicit casting depends on whether the conversion is considered safe or might result in
        data loss or unexpected behavior.
        
        Implicit casting is performed automatically by the compiler when there is no risk of data loss. For example, converting an int to a long.
    
            int intValue = 42;
            long longValue = intValue; // Implicit casting from int to long
        
        In this case, the compiler allows the conversion from int to long without requiring any explicit casting syntax.

2. Explicit Casting (Narrowing Conversion):

        Explicit casting is required when there is a possibility of data loss, and the programmer needs to explicitly indicate
        the conversion. This is done using the cast operator. For example, converting a double to an int:
    
            double doubleValue = 123.45;
            int intValue = (int)doubleValue; // Explicit casting from double to int
        
        Here, the (int) is used to explicitly cast the doubleValue to an int. Note that this may result in the loss of the
        fractional part of the number.
