Variables

        In C#, variables are used to store and manipulate data in a program. Each variable has a specific data type that
        determines the kind of data it can hold. Here's an overview of variables in C#:
        
        
1.Variable Declaration

        To declare a variable, you specify its data type followed by a name. For example:
     
       

        int age; // Declaring an integer variable
        double salary; // Declaring a double-precision floating-point variable
        string name; // Declaring a string variable
        bool isStudent; // Declaring a boolean variable
       

     
        
        
2.Variable Initialization:

        After declaring a variable, you can initialize it with a value using the assignment operator (`=`):
      
       

        age = 25; // Initializing the 'age' variable
        salary = 50000.50; // Initializing the 'salary' variable
        name = "John Doe"; // Initializing the 'name' variable
        isStudent = true; // Initializing the 'isStudent' variable
       

        
        
3.Combining Declaration and Initialization:

        You can combine the declaration and initialization in a single line:
      
        

        int age = 25; // Combining declaration and initialization
        double salary = 50000.50; // Combining declaration and initialization
        string name = "John Doe"; // Combining declaration and initialization
        bool isStudent = true; // Combining declaration and initialization
        

     
        
        
4.Variable Naming Rules:

       

        - Variable names must start with a letter or an underscore.
        - Subsequent characters can be letters, digits, or underscores.
        - C# is case-sensitive, so `myVariable` and `MyVariable` are different variables.
        - Avoid using reserved keywords (e.g., `int`, `string`) as variable names.
       

        
        
5.Implicit and Explicit Typing:

        You can use implicit typing (`var` keyword) when the compiler can infer the type:
       
        

            var count = 10; // Implicitly typed as an integer
            var price = 29.99; // Implicitly typed as a double
        

   
        
        
6.Constants

        Constants are variables whose values cannot be changed once set. They are declared using the `const` keyword:
      
       
 const double Pi = 3.14159;

       
        
        
7.Dynamic Typing:

        C# also supports dynamic typing using the `dynamic` keyword. The type is resolved at runtime:
     
        

       
        
        
8.Nullable Types:

        Nullable types allow variables to have a value of `null` in addition to their regular data type:
      
        

            int? nullableInt = null; // Nullable integer
            double? nullableDouble = 3.14; // Nullable double
        

     
        
9.Scope of Variables:

        Variables have a scope, and they can be local to a block, a method, or be members of a class.
        
        
10.Value Types vs. Reference Types:

        Value types store their data directly, while reference types store a reference to the memory location where the data is
        stored.
        
        Understanding and effectively using variables is fundamental to programming in C#. They allow you to store and
        manipulate data, making your programs dynamic and responsive to different inputs.
