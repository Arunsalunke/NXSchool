Console

        In C#, the Console class is commonly used to interact with the console and display output. Here are some examples of how
        you can output information to the console:
        
       
 1. Printing Text:

       
       

        Console.WriteLine("Hello, World!");   
       

       This line of code prints "Hello, World!" followed by a newline to the console.
      
        
        
2. Printing Variables:

       
        

            string name = "John";
            int age = 25;
            Console.WriteLine($"Name: {name}, Age: {age}");
        

        This uses string interpolation to include variable values within a string.
        
        
3. Formatting Output:

    

        int number = 42;
        Console.WriteLine("The answer to the ultimate question is: {0}", number);
    
 
        This uses placeholder {0} to format the output with the value of number.
        
       
4. Reading Input:
 
       

        Console.Write("Enter your name: ");
        string userName = Console.ReadLine();
        Console.WriteLine($"Hello, {userName}!");
        Console.ReadLine() reads a line of text from the console.
       

        
        
       
5. Output Without Newline:
 
      
       

        Console.Write("This ");
        Console.Write("is ");
        Console.Write("on ");
        Console.Write("the ");
        Console.Write("same ");
        Console.Write("line.");
       

        Using Console.Write to print on the same line without adding a newline character.
        

        
6. Formatted Numbers:

       
        

            double price = 42.99;
            Console.WriteLine($"Price: {price:C2}");
        

        This formats the price as currency with two decimal places.
        
       
7. Custom Formatting:
 
        

            DateTime currentDate = DateTime.Now;
            Console.WriteLine($"Current Date: {currentDate:yyyy-MM-dd HH:mm:ss}");
        

        
        Custom formatting for displaying the current date and time.
        
        These are just a few examples of how you can use the Console class to output information in C#. The key is to use
        Console.WriteLine for adding a newline after the output and Console.Write for outputting without a newline.
